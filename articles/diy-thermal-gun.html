DIY Temperature Thermal Gun

A non-contact temperature project that uses an infrared sensor, Arduino, and display to explore how surface temperature can be measured.

Completed Prototype 6 min read Student project note

Materials

  • Arduino UNO or compatible board
  • Infrared temperature sensor such as MLX90614 or AMG8833
  • OLED or LCD display
  • Jumper wires
  • Battery pack or USB power
  • Optional enclosure
  • Optional laser pointer for aiming

Tools

  • Arduino IDE
  • Computer
  • Basic hand tools
  • Soldering iron if building a permanent version
  • Multimeter if available

Optional upgrades

  • Bluetooth or Wi-Fi module
  • Data logging
  • Mobile or web dashboard
  • Cleaner enclosure

Project overview

The thermal gun uses an infrared temperature sensor connected to an Arduino. The sensor reads emitted infrared radiation from a surface, and the Arduino displays the temperature on a small screen.

Status note: This is a learning prototype. It is not a medical thermometer and should not be used to diagnose fever or illness.

Problem being solved

Temperature can be useful to measure without contact, especially for surfaces, electronics, appliances, or quick experiments. Commercial devices hide the electronics, so building a prototype makes the measurement process easier to understand.

Why I built it

I built this to learn how infrared sensors work and how a microcontroller converts sensor data into a readable temperature value.

How it works

Objects emit infrared radiation based on their temperature. The sensor detects that radiation and sends a reading to the microcontroller. The Arduino processes the value and displays it as temperature.

Step-by-step build process

  1. Gather the Arduino, infrared sensor, display, wires, and power source.
  2. Check the sensor datasheet or library documentation for the correct wiring.
  3. Connect the sensor to the Arduino using the required communication pins.
  4. Connect the display module.
  5. Install the needed Arduino library for the sensor.
  6. Upload a basic example sketch and check readings on Serial Monitor.
  7. Test the sensor on objects with stable temperatures.
  8. Compare readings with a known thermometer for a rough check.
  9. Mount the sensor so it has a clear line of sight.
  10. Document limitations such as distance, surface material, and room conditions.

Calibration or testing

Testing should include objects at room temperature and slightly warmer or cooler objects. Readings can change based on distance, surface color, and emissivity, so the prototype should be treated as a learning tool.

Challenges faced

  • Different surfaces can give different readings even when they feel similar.
  • The sensor needs a clear view of the target.
  • Aiming and distance matter more than I expected.

What I learned

  • Non-contact measurement is useful but sensitive to setup.
  • A sensor reading needs context before it can be trusted.
  • Health-related prototypes need careful safety language.

Future improvements

  • Add a better enclosure with a clear display window.
  • Add data logging for repeated measurements.
  • Add calibration options for different surface types.
  • Compare different infrared sensor modules.

Safety or accuracy note

This prototype is for learning and demonstration only. It is not a medical device and should not be used for diagnosis or treatment.
