# Image Attributions

This version does not use downloaded third-party photos or hotlinked images.

All visuals used on the site are custom SVG diagrams created for this project and stored in `assets/diagrams/`, plus the custom `favicon.svg` logo in the site root.

Because the diagrams are original project assets, there are no external image licenses or attribution requirements in the current build.

If future photos are added, record each one here with:

- File name
- Source URL
- Creator
- License
- Required attribution text
- Date downloaded

Do not use random Google Images results. Use only original photos, properly licensed images, or public domain assets.
